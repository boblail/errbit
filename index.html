Errbit by boblail

The open source error catcher that's Hoptoad API compliant

This is a fork

This is a fork of github.com/errbit/errbit.

Changes

The major changes I've made on boblail/errbit are:
  1. Added the ability to apply bulk actions to selected errs
  2. Added the ability to merge errs
  3. Tightened up the user interface
  4. Made column headers sortable
  5. Enabled Errbit to receive notifications from the JavaScript notifier
  6. Worked on improving performance through caching values onto Problem
  7. Notice summary screens show breakdowns for certain values
  8. Refactored notice API so that it is (a) easier to test and (b) possible to re-report all notices
